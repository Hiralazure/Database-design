# Database Design

## Overview

This repository contains multiple database design assignments showcasing various database design concepts, patterns, and real-world scenarios. Each assignment includes detailed file definitions and Entity-Relationship (ER) diagrams.

## Contents

- **Database Design Assignments**: Collection of diverse database design projects
- **File Definitions**: Detailed documentation of each file's structure and purpose
- **ER Diagrams**: Visual representations of database relationships and schemas
- **Schema Definitions**: Complete database schema files for each assignment

## Assignment Structure

Each assignment folder contains:

- `definition.md` - Complete file and table definitions
- `er-diagram.md` or `diagram.png` - ER diagram(s) showing relationships
- `er-diagram.png` or `diagram.png` - png diagram(s) showing relationships
- `schema.txt` - text file containing SQL script with table creation statements
